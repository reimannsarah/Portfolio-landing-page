PROJECTS

  • S&R Furry Friends

    A website for an animal shelter called S&R's Furry Friends. This project includes 4 pages that are linked in the navigation bar. The each of the pages were built using HTML and CSS. Take a look at the Available Now page to view the animals waiting to find their forever homes.

    View Page

  • Berli Designs

    A webpage for an interior design company called BERLI Designs. This page was built using HTML and CSS. It features several flexboxes and box model styling in various places. BERLI Designs can take your outdated hotel lobby from drab to fab.

    View Page

  • Flashie's Cupcakes

    This project is the home page for Flashie's Cupcakes. It was built using HTML and CSS styling. It includes lists of cupcake flavors as well as coffee offerings. Prices are high to account for the ingredients being imported from all over the world. The famous Coco Nono is baked right inside the shell of a coconut.

    View Page

  • Buckets and Opal Blog

    The homepage of blog for Buckets and Opal, a mother/daughter hairless guinnea pig duo. The page was built using HTML and CSS styling. You'll see two blog posts written by Opal describing the trials and tribulations of the two skinny pigs.

    View Page
